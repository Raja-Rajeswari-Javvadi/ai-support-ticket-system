AI Support Ticket Classification System
Note:
Create a .env file and add your Gemini API key before running:

GEMINI_API_KEY=your_key_here


This project is an AI-powered support ticket system built using Django REST Framework and Gemini LLM.

Features:
- Create support tickets
- AI auto-detect category
- AI assign priority
- Ticket listing & filtering
- Stats API

Tech Stack:
- Python
- Django REST Framework
- Gemini API
- SQLite
- HTML/CSS/JS

Run Instructions:

pip install -r requirements.txt
python manage.py runserver

Open:
http://127.0.0.1:8000
